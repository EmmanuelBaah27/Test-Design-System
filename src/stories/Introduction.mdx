# Mande Design System

Welcome to the **Mande Design System** — a component library and design documentation for Mande products.

## Overview

This design system is built with:

- **Atomic Design** — Components are organized as Atoms, Molecules, and Organisms
- **Tailwind CSS v4** — With design tokens for colors, typography, spacing, and radius
- **React 19** — Modern React with TypeScript
- **Next.js** — For the documentation site and Storybook integration

## Structure

| Level | Description | Examples |
|-------|-------------|----------|
| **Atoms** | Basic building blocks | Button, Input, Label, Chip |
| **Molecules** | Combinations of atoms | InputWithLabel, FormField |
| **Organisms** | Complex UI sections | Dialog, Dropdown, Card |

## Design Tokens

All design decisions are driven by tokens defined in `globals.css`:

- **Typography** — H1–H3, xlg, lg, base, small (with responsive scaling)
- **Colors** — Neutral, Primary, Teal, Success, Negative, Warning, Blush, Info
- **Spacing** — 3xs through 9xl
- **Radius** — none through full
