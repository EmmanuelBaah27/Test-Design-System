# Design Tokens

Mande Design System uses CSS custom properties for all design decisions.

## Colors

### Neutral
| Token | Usage |
|-------|-------|
| `neutral-white` | Backgrounds, surfaces |
| `neutral-50` – `neutral-900` | Text, borders, fills |
| `neutral-black` | Primary text |

### Semantic
| Palette | Usage |
|---------|-------|
| **Primary** | Brand, CTAs |
| **Teal** | Links, accents |
| **Success** | Positive states |
| **Negative** | Errors, destructive |
| **Warning** | Caution states |
| **Info** | Informational |
| **Blush** | Soft accents |

## Typography

| Scale | Size | Weight |
|-------|------|--------|
| H1 | 28px (desktop) | 600 |
| H2 | 24px | 600 |
| H3 | 20px | 600 |
| xlg | 18px | 400/500/600 |
| lg | 16px | 400/500/600 |
| base | 14px | 400/500/600 |
| small | 12px | 400/500/600 |

## Spacing

`spacing-3xs` (2px) → `spacing-9xl` (80px)

## Radius

`radius-none` (0) → `radius-full` (1000px)
